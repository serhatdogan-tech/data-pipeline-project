# ETL Pipeline – Svensk Befolkningsdata

## Projektöversikt
En end-to-end ETL-pipeline som hämtar öppen befolkningsdata från SCB 
(Statistiska Centralbyrån), bearbetar den med Python och lagrar 
resultatet strukturerat i en SQLite-databas.

## Teknologier
- **Python 3.9** – Databearbetning och pipeline-logik
- **Pandas** – Datatransformation
- **SQLite** – Lokal datalagring
- **Git/GitHub** – Versionshantering
